# Redbooks 小红书爬虫 - OpenClaw 使用说明书

## 技能概述

这是一个功能完整的小红书笔记爬虫工具，支持关键词搜索、主页推荐、博主主页爬取，具有图形化界面。

**位置**: `/home/qian/Desktop/redbooks/`

**主程序**: `/home/qian/Desktop/redbooks/crawler_ultimate.py`

## 快速启动

```bash
cd /home/qian/Desktop/redbooks
python3 crawler_ultimate.py
```

## 支持的命令格式

### 1. 基础爬取命令

**格式**: `爬取 [关键词] [数量]`

**示例**:
- `爬取 护肤品 10` - 爬取10条关于护肤品的笔记
- `爬取 美食 20` - 爬取20条关于美食的笔记
- `爬取 旅游 5` - 爬取5条关于旅游的笔记

### 2. 高级命令格式

**格式**: `爬取 [关键词] [数量] [模式] [选项]`

**参数说明**:
- `关键词`: 搜索关键词（必填）
- `数量`: 爬取笔记数量，默认30
- `模式`:
  - `标准` - 完整数据采集（含评论、图片）
  - `极速` - 快速采集基础信息
- `选项`:
  - `只要文本` - 只获取文本内容
  - `只要图片` - 只下载图片
  - `只要视频` - 只下载视频
  - `不下图片` - 不下载图片

**示例**:
- `爬取 护肤 10 标准 只要文本`
- `爬取 美食 20 极速 不下图片`
- `爬取 旅行 15 标准 只要图片`

### 3. 主页推荐爬取

**格式**: `爬取主页推荐 [数量]`

**示例**:
- `爬取主页推荐 20` - 爬取首页推荐20条

### 4. 博主主页爬取

**格式**: `爬取博主 [博主主页链接] [数量]`

**示例**:
- `爬取博主 https://www.xiaohongshu.com/user/profile/xxxxx 10`

### 5. 导出命令

**格式**: `导出 [格式]`

**示例**:
- `导出 Excel` - 导出为Excel文件
- `导出 CSV` - 导出为CSV文件

## 数据字段说明

爬取后可获取以下字段：

| 字段 | 说明 |
|------|------|
| title | 标题 |
| author | 作者 |
| content | 正文内容 |
| tags | 标签（#xxx） |
| publish_time | 发布时间 |
| ip_region | IP地区 |
| like_count | 点赞数 |
| collect_count | 收藏数 |
| comment_count | 评论数 |
| note_type | 笔记类型（图文/视频） |
| note_link | 笔记链接 |
| image_urls | 图片链接 |
| video_url | 视频链接 |
| comments | 评论列表 |

## 使用流程

1. **启动程序**: `python3 /home/qian/Desktop/redbooks/crawler_ultimate.py`
2. **登录账号**: 首次使用需要在弹出的浏览器中登录小红书
3. **执行爬取**: 使用上述命令格式进行爬取
4. **导出数据**: 爬取完成后导出结果

## 注意事项

1. 首次使用需要登录小红书账号
2. 建议适当设置爬取间隔，避免被限制
3. 程序启动后会显示GUI界面，可在界面中操作或通过命令控制
4. Cookie会自动保存，下次使用无需重新登录
5. 仅供学习研究使用，请遵守小红书的服务条款

## 常见问题

**Q: 程序卡在笔记页面不动？**
A: 这是正常现象，程序正在提取数据，请耐心等待。

**Q: 图片下载失败？**
A: 可能是网络问题或图片需要认证，不影响文本数据获取。

**Q: 如何查看爬取结果？**
A: 在GUI界面的"结果"选项卡中查看，或使用导出命令导出。

**Q: 如何停止爬取？**
A: 点击GUI界面上的"停止"按钮，或关闭程序窗口。
